Examples of Coperr programs
===========================

Hello World
^^^^^^^^^^^

Hello world in Coperr:

::

    print("Hello, World!");

This outputs:

::

    Hello, World!

Math Examples
^^^^^^^^^^^^^

Some examples of how math works (and order of operations):

::

    print(10/23);
    print(1+2);
    print(10-2);
    print(100/2);
    print(33*2);
    print(10%3);
    print(10+3/2);

Outputs:

::

    0.43478260869565216
    3
    8
    50.0
    66
    1
    11.5
